# 🏍️ WebWise Mechanics — Responsive Business Website

## 📌 Overview

WebWise Mechanics is a responsive business website designed for a **bike repair and service workshop**.
The goal of this project is to showcase how Bootstrap can be used to build a real-world, professional website.

---

## 🎯 Features

* Fully responsive design across all devices
* Clean and modern light theme with retro-inspired colors
* Multi-page structure:

  * Home (services, clients, testimonials)
  * About (company details, mission)
  * Contact (form + business info)

---

## 🧩 Key Sections

### 🏠 Home Page

* Hero section
* Services (cards)
* Client logos
* Image carousel (auto-sliding)
* Customer testimonials

### ℹ️ About Page

* Company introduction
* Mission & experience
* Why choose us section

### 📞 Contact Page

* Responsive contact form
* Workshop details
* User-friendly layout

---

## 🛠️ Tech Stack

* Responsive Grid System
